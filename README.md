# vue-book

> A Vue.js project  实现图书增删改查

## 技术栈
- vue
- vue-router
- vuex
- axios
- nodejs

## 项目启动

``` bash
# 安装依赖
npm install

#起服务
mock文件下 命令行 node server

# 8080端口热加载
npm run dev

```
